# **App Name**: Equation Ace

## Core Features:

- Equation Input: Allow users to input equations using a LaTeX-based editor.
- Formula Recognition: Utilize AI to recognize handwritten or image-based formulas.
- PDF Export: Convert equations and notes into PDF format for easy sharing and printing.
- Equation Library: Create and manage a personal library of equations for quick access and reuse.
- Unit Conversion: AI-powered tool to automatically identify and convert units within equations.

## Style Guidelines:

- Primary color: Deep Blue (#1A237E) to inspire trust, enhance focus, and improve readability on a screen.
- Background color: Very light desaturated blue (#E8EAF6). Using the same hue as the primary color but in desaturated form helps unify the look.
- Accent color: Violet (#7B1FA2), a distinctive color, but close enough in hue to the primary color to maintain harmony.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text.
- Use clear, mathematical symbols and icons to represent different equation categories and functions.
- A clean, intuitive layout that prioritizes equation readability and ease of navigation.
- Subtle transitions and animations to provide feedback and guide the user through the equation input process.
